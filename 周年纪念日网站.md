# 周年纪念日前端网站

## 需求

### 时间轴

- 有个时间轴，能够记录时间点，当前时间点的说明、图片等
- 时间点的时间、说明、图片可以配置
- 使用鼠标滚轮可以翻动时间轴，向上即时间轴向前，向下即时间轴向后
- 翻动时间轴时，显示对应的时间点说明及图片

### 信封

- 当拉到网页最下方，有一个信封
- 使用鼠标打开信封后是一段文字，文字可以配置

### 其他

- 网站不需要后端的参与
- 网站可以配置背景音乐
- 网站色彩样式需要浪漫、温馨的氛围
- 后面可以上传到GitHub Pages用于生成专属的域名链接

## 具体实现方式参考

1. 技术栈建议

- **框架**: Vue.js 或 React（方便数据配置）。
- **动画**: Framer Motion (React) 或 GSAP。GSAP 在处理“随滚动条翻动”的动画上非常强大。
- **样式**: Tailwind CSS。可以轻松调出“莫兰迪色系”或“粉嫩温馨”的配色。
- **音乐**: 使用 HTML5 `<audio>` 标签。
- 核心配置 (JSON 数据结构)

你可以把所有内容放在一个 `config.js` 中，方便以后修改：

javascript

```
const config = {
  bgMusic: "https://example.com",
  themeColor: "#fce4ec", // 浅粉色
  timeline: [
    { date: "2023-05-20", title: "第一次见面", desc: "那天阳光很好...", img: "pic1.jpg" },
    { date: "2023-12-25", title: "第一个圣诞节", desc: "我们去了...", img: "pic2.jpg" },
    // 更多时间点...
  ],
  letter: "亲爱的，这封信是写给你的..."
};
```

Use code with caution.

3. 核心功能实现思路

**A. 滚轮控制时间轴**

利用 **Vertical Snap Scrolling (竖向贴合滚动)**。

- 每一条时间线记录占满一屏（100vh）。
- 使用 `scroll-snap-type: y mandatory` 让鼠标滚动时自动“吸附”到下一个时间点。

**B. 互动信封 (CSS 实现)**

信封可以用简单的 CSS 绘图实现。点击时触发一个 `open` 状态：

- **盖子 (Flap)**: 使用 `transform: rotateX(180deg)` 向上翻开。
- **信纸 (Paper)**: 在盖子翻开后，通过 `z-index` 变化和位移动画（向上平移）拉出文字。

**C. 浪漫氛围感设计**

- **背景**: 使用 `linear-gradient` 渐变色，或者添加一些飘落的樱花/爱心粒子效果（推荐使用 `tsparticles` 库）。
- **字体**: 引入 Google Fonts 的手写体（如 *Dancing Script*）来展示信件内容。
- 部署到 GitHub Pages

1. 在 GitHub 新建仓库。
2. 将代码推送到仓库。
3. 在仓库设置 (**Settings > Pages**) 中，选择 `main` 分支作为部署来源。
4. 你的网站就会在 `https://github.io` 上线。
